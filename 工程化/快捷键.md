## VSCode

### 光标移动

- 针对单词的光标移动
option + ←/→

- 光标移动到行首或者行末
command + ←/→

- 代码块的光标移动{} /在一对花括号之间跳转
command + shift + \

### 多光标

- 多行编辑
  command + option + ↑/↓ 或者 shift + option + 鼠标拖动
        
- 选中全部与当前所选内容相同的文本
  shift + command + L
        
- 在点击位置添加光标 
  option + click
  
- 选中下一个与当前所选内容相同的文本 
  command + D
